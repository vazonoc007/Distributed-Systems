Виведення привітання

1. При запуску програма виводить повідомлення:

Hello World!

2. Введення імені користувача

Програма пропонує користувачу ввести своє ім’я.

Після введення виводиться персоналізоване привітання:

Hello, <ім’я>!

3. Обчислення довжини імені та факторіалу

Програма визначає кількість символів у введеному імені.

Обчислює факторіал цього числа.

Виводить результат у форматі:

Your name has N letters. N! = результат

4. Введення дати народження

Користувач вводить дату народження у форматі:

DD.MM.YYYY

5. Обчислення віку

Програма визначає поточну дату.

Обчислює вік користувача в роках.

Додатково може обчислювати вік у днях.

6. Виведення результату

Результат подається у вигляді:

Today is <поточна дата>, you are X years (Y days) old.
