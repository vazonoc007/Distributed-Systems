# MQ-server
Дана програма реалізує консольний чат-клієнт, який використовує технологію черг повідомлень для обміну даними між користувачами. 
Взаємодія з сервером здійснюється через систему повідомлень, що забезпечує надійну доставку та асинхронну обробку даних.

Програма підтримує такі команди:

1. ping

Використовується для перевірки доступності сервера.

2. echo <text>

Надсилає текстове повідомлення та отримує його назад.

3. login <login> <password>

Виконує авторизацію користувача.

4. list

Отримує список активних користувачів.

5. msg <user> <text>

Надсилає повідомлення іншому користувачу.

6. file <user> <filename>

Передає файл іншому користувачу.

7. exit

Завершує роботу програми та закриває з’єднання.

8. AFK (Away From Keyboard)

Якщо користувач не виконує жодних дій протягом певного часу.
