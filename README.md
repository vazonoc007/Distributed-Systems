# RMI-server
Дана програма реалізує консольний клієнт для обміну повідомленнями з використанням технології віддаленого виклику методів (RMI). 
Вона дозволяє взаємодіяти із сервером шляхом виклику його методів так, ніби вони виконуються локально.

Програма підтримує такі команди:

1. ping

Викликає відповідний метод сервера для перевірки доступності.

2. echo <text>

Надсилає текст на сервер та отримує його назад.

3. login <login> <password>

Виконує авторизацію користувача та отримує ідентифікатор сесії.

4. list

Отримує список активних користувачів.

5. msg <user> <text>

Надсилає повідомлення іншому користувачу.

6. file <user> <filename>

Передає файл іншому користувачу.

7. exit

Завершує роботу програми та закриває з’єднання.
