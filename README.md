# TCP-Sockets
Дана програма реалізує консольний клієнт для обміну повідомленнями через мережу з використанням TCP-сокетів. 
Вона забезпечує підключення до сервера, обмін даними відповідно до заданого протоколу та обробку команд користувача.

Програма підтримує такі команди:

1. ping

Перевіряє доступність сервера

Отримує відповідь підтвердження

2. echo <text>

Надсилає текст на сервер

Отримує і виводить той самий текст

3. login <login> <password>

Виконує авторизацію користувача

Обробляє можливі помилки входу

4. list

Отримує список активних користувачів

Виводить їх у консоль

5. msg <user> <text>

Надсилає повідомлення іншому користувачу

Отримує підтвердження від сервера

6. file <user> <filename>

Передає файл іншому користувачу

Дані файлу перетворюються у масив байтів

7. receive msg

Періодично перевіряє наявність нових повідомлень

Виводить отримані повідомлення

8. receive file

Перевіряє наявність файлів

Отримує та зберігає їх

9. exit

Завершує роботу програми

Закриває з’єднання з сервером
